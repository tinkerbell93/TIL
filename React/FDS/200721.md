## React Component 정의

컴포넌트 문법

hook을 배우기 이전에 생각해야하는것은 컴포넌트안에 상태가 있는지 없는지 확인하고 컴포넌트에 상태가 있으면 class를 사용해야하고, 없을때 라이프사이클이 있다면 class 없으면 function을 사용한다.

hook을 배운 후는 class를 배운 컴포넌트를 function으로 바꾸는 추세이기 때문에 function을 사용한다.

- **클래스 컴포넌트**

  ```react
  // class 컴포넌트
  // react의 기능을 다 가져와서 나의 클래스 컴포넌트가 됨
  // 하나의 컴포넌트를 하나의 디폴트를 컨벤션으로 가지고 있기위해 default를 사용
  export default class ClassComponent extends React.Component {
    // class를 상속받으면 render를 구현해야한다.
    // render를 오버라이드를 해야함
    render() {
      return (
        <div>
          <h1>나는 클래스 컴포넌트얌</h1>
        </div>
      );
    }
  }
  ```

  

- **함수형 컴포넌트**

  **[일반 함수]**

  ```react
  import React from 'react';
  
  export default function FunctionComponent() {
     return (
       <div>
         <h1>나는 함수형 컴포넌트</h1>
       </div>
     )
  }
  ```

  **[화살표 함수]**

  ```react
  import React from 'react';
  
  const FunctionComponent = () => (
    <div>
        <h1>나는 함수형 컴포넌트</h1>
    </div>
  );
  
  export default FunctionComponent;
  ```

  

## Props

부모가 자식한테 props를 넣어준다.

- **클래스 컴포넌트**

  ```react
  export default class ClassComponent extends React.Component {
    // 기본 props 설정
    static defaultProps = {
      age: 0,
    };
  
    render() {
      // 구조할당으로 변수에 선언하여 사용
      const { name, age, children } = this.props;
      return (
        <div>
          <h1>나는 클래스 컴포넌트야</h1>
          {name && <h3>나는 {name}이야</h3>}
          {age !== undefined && <h3>나는 {age}살이야</h3>}
          {children && <h3>나는 {children}이 있어</h3>}
        </div>
      );
    }
  }
  ```

  

- **함수형 컴포넌트**

  **[화살표 함수]**

  ```react
  // 매개변수로 props 주입
  const FunctionalComponent = ({ name, age, children }) => (
    <div>
      <h1>나는 함수형 컴포넌트야</h1>
      {name && <h3>나는 {name}이야</h3>}
      {age !== undefined && <h3>나는 {age}살이야</h3>}
      {children && <h3>나는 {children}이 있어</h3>}
    </div>
  );
  
  export default FunctionalComponent;
  
  // 기본 props 설정
  FunctionalComponent.defaultProps = {
    age: 0,
  };
  ```

  

## state

외부에 영향을 받지 않고 본인의 값으로 렌더를 다시 한다.

state는 객체다.

```react
// nop
state = 0 

// yes
state = {
  count: 0
}

// yes
constructor(props) {
  
}
```



- **클래스 컴포넌트**

  ```react
  import React from 'react';
  
  export default class ClassComponent extends React.Component {
    state = {
      count: 0,
    };
  
    // 라이프사이클 훅
    componentDidMount() {
      console.log('componentDidMount');
  
      setInterval(() => {
        // 1. 새로운 setState 객체를 넣는 방법
        // 과거 state값과 상관 없을 때 사용
        this.setState({ count: this.state.count + 1 });
  
        // 2. setState에 함수를 넣는 방법
        // 기존 stete의 값에 조합하여 새로운 state를 만들 경우 사용
        this.setState((state) => ({ count: state.count + 1 }));
      }, 1000);
    }
  
    render() {
      return (
        <div>
          <h1>
            클래스 컴포넌트 <br /> {this.state.count}
          </h1>
        </div>
      );
    }
  }
  ```

  **[constructor로 state 초기값 설정]**

  ```react
    constructor(props) {
      super(props);
      this.state = { count: 0 };
    }
  ```

```react
function App() {

```
