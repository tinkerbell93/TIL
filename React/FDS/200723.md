# 리액트 라우팅 이해하기

서버에 가지 않고 클라이언트 내부에서 다른 컴포넌트를 보여준다.

리액트는 이것에 대해 관여하지 않는다. 

리액트 라우터 돔이라서 컴포넌트를 준다. 우리는 컴포넌트를 사용하면 되기때문에 props를 알아야한다.

### Static 라우팅

1. `npm i react-router-dom`

2. 정의

   - / => home
   - /profile => profile
   - /about => about

   ```react
   <BrowserRouter>
     <Route component={Home} />
     <Route component={Profile} />
     <Route component={About} />
   </BrowserRouter>
   ```
